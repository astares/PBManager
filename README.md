# PBManager
Pharo Bootstrap manager with UI

## Quick Start

Fist load Espell as a prerequisite

```Smalltalk
Metacello new 
	repository: 'github://carolahp/espell';
	baseline: 'Espell';
	load 
```


```Smalltalk
Metacello new 
	repository: 'github://carolahp/PBManager';
	baseline: 'PBManager';
	load 	
```
